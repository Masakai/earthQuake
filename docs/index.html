Raspberry Shake 4D + JMA Algorithm · v1.5.0

リアルタイム震度モニター
ブラウザで見る

Raspberry Shake 4D の波形データをリアルタイムに受信し、気象庁の計測震度アルゴリズムで震度を算出。ブラウザから地図つきで確認できます。

GitHub で見る しくみを読む
Features

主な機能

センサーから画面まで、すべてオープンソース。

📡

UDP リアルタイム受信

Raspberry Shake の DATACAST を直接受信。遅延なく波形データを取得します。

📊

JMA 計測震度アルゴリズム

気象庁の公式フィルター処理・閾値・計算式をそのまま実装。I値・加速度（gal）をリアルタイム表示。

🗺️

震源地図（Leaflet）

P2P地震情報から取得した最新地震を地図上にマッピング。マーカーサイズはマグニチュードに比例。

🔔

音声アラート

震度に応じた音声読み上げ（macOS say -v Kyoko）。計測震度が震度1相当（I≥0.5）を超えた時点で発話し、揺れが強まれば言い直します。

⚡

緊急地震速報（参考）

P2P地震情報 WebSocket API から EEW を受信。バナーで赤点滅表示。

📈

I値・STA/LTA 推移グラフ

Chart.js による 2 軸グラフで、直近 5 分の計測震度と STA/LTA 値の変化を可視化。

🖥️

TUI ダッシュボード

ブラウザ不要のターミナル版も付属。SSH 接続のまま監視できます。

🧪

UDP シミュレーター

実機なしでも動作確認できる合成波形送出ツールを同梱。

🔬

ワンクリック波形解析

P2P地震情報テーブルの「解析」ボタンを押すだけで、自局波形のダウンロードとスペクトログラム解析を自動実行。結果画像をモーダル表示します。

How it works

データの流れ

センサーの生波形が、震度数値としてブラウザに届くまで。

1

波形受信

Raspberry Shake が UDP DATACAST で生波形カウント値を送出

2

ガル変換

感度値で除算し、加速度（gal = cm/s²）に変換

3

JMA フィルター

気象庁規定の周波数フィルターを適用して人体感覚に補正

4

震度算出

I = 2 log₁₀(a) + 0.94 で計測震度 I 値を計算

5

WebSocket配信

FastAPI が 1 秒ごとに JSON をブラウザへブロードキャスト

→ アルゴリズムの詳しい解説を読む

Requirements

必要なもの

ハードウェアと基本的なソフトウェアが揃えば動き始めます。

ハードウェア

  • Raspberry Shake 4D（RS4D）
  • 同一ネットワーク上の Mac または Linux PC

ソフトウェア

  • Python 3.10 以上
  • macOS（音声アラートに say を使用・macOS 専用）

Python パッケージ

  • numpy / scipy / obspy
  • rich（TUI 版）
  • fastapi / uvicorn / jinja2（Web 版）
  • websocket-client / matplotlib
Quick Start

使い始め方

クローンから起動まで 5 分で完了します。

本ソフトウェアが算出する計測震度は参考値です。公式な震度情報ではありません。
1

クローン＆仮想環境を作成

git clone https://github.com/Masakai/earthQuake.git
cd earthQuake
python3 -m venv .venv
source .venv/bin/activate
pip install numpy scipy obspy rich websocket-client fastapi uvicorn jinja2 matplotlib
2

Raspberry Shake の DATACAST を設定

Shake の Web 設定画面で UDP/DATACAST を有効にし、送信先 IP とポート（既定: 8888）を入力します。

3

Web ダッシュボードを起動

bash scripts/start_web.sh --station R38DC
# ブラウザで http://localhost:8080 を開く
4

実機なしでシミュレーターを使う

# ターミナル 1
bash scripts/start_web.sh --station R38DC --bind 127.0.0.1:9999

# ターミナル 2
.venv/bin/python3 src/simulate_udp.py --intensity 3.0 --dest 127.0.0.1:9999
